# 밥빵 (BabPbang) 화면 설계서

> 촬영현장 식사 조율 앱 — 함께 먹을지 결정하고, 식당 추천받고, 각자 메뉴까지 고르는 앱

---

## 앱 개요

| 항목 | 내용 |
|------|------|
| 대상 사용자 | 촬영현장 스태프 (조명팀, 카메라팀, 연출팀 등) |
| 핵심 문제 | 식사시간마다 "같이 먹을래요?" "어디 갈까요?" "뭐 먹을래요?" 를 매번 물어보는 번거로움 |
| 해결 방식 | 방 생성 → 참여 투표 → 식당 추천 → **식당 선정 (투표 or 리더 직접 선택)** → 메뉴 선택 → 요약 확인 |

---

## 사용자 흐름 (User Journey)

```
[앱 진입]
    ↓
[온보딩: 이름 + 소속팀 입력] — 최초 1회
    ↓
[홈 화면]
    ├── 새 밥약속 만들기 → [밥약속 생성 (식당 선정 방식 선택)] → [초대 링크 공유]
    └── 초대 링크 수신 → [참여 여부 투표]
                              ↓ (먹을게요!)
                        [식당 추천 목록]
                              ↓
                    ┌─────────────────────┐
                    │  식당 선정 방식      │
                    ├─────────────────────┤
                    │ A) 투표 모드         │
                    │    식당 상세 열람     │
                    │    → "이 식당 추천!" │
                    │    → 투표 결과 화면  │
                    │                     │
                    │ B) 리더 직접 선택    │
                    │    식당 상세 열람     │
                    │    → "여기서 먹어요" │
                    │      (리더만 활성화) │
                    └─────────────────────┘
                              ↓ (식당 확정)
                        [메뉴 선택]
                              ↓
                        [최종 요약 화면]
```

## 화면 목록

| # | 화면 이름 | 경로 | 주요 기능 |
|---|---------|------|---------|
| 1 | 온보딩 | `/onboarding` | 이름, 소속팀 입력 |
| 2 | 홈 | `/` | 진행 중인 밥약속, 새 약속 만들기 |
| 3 | 밥약속 만들기 | `/room/new` | 식사 시간, 인원 설정, **식당 선정 방식 선택** |
| 4 | 초대 공유 | `/room/[id]/invite` | 링크/코드 공유 |
| 5 | 참여 여부 투표 | `/room/[id]/vote` | 참여/불참 선택 |
| 6 | 식당 추천 목록 | `/room/[id]/restaurants` | 위치 기반 식당 추천 |
| 7A | 식당 상세 (투표 모드) | `/restaurant/[id]?mode=vote` | 식당 정보 + "이 식당 추천해요!" 투표 버튼 + 실시간 투표 현황 |
| 7B | 식당 상세 (리더 선택 모드) | `/restaurant/[id]?mode=leader` | 식당 정보 + 리더만 보이는 "여기서 먹어요!" 확정 버튼 |
| 8 | 메뉴 선택 | `/room/[id]/menu` | 각자 메뉴 고르기 |
| 9 | 최종 요약 | `/room/[id]/summary` | 참여자별 메뉴 확인 |

## Navigation Flow

```
[온보딩 (/onboarding)]
  └── 완료 → [홈 화면 (/)]

[홈 화면 (/)]
  ├── "새 밥약속 만들기" 탭 → [밥약속 만들기 (/room/new)]
  │     └── 완료 → [초대 공유 (/room/[id]/invite)]
  │           └── 링크 복사 후 대기 → [참여 현황 실시간 확인]
  │                                       └── 모두 응답 or 시간 되면 → [식당 추천 목록]
  │
  ├── 진행 중인 약속 카드 클릭 → 현재 단계에 맞는 화면으로 이동
  └── 초대 링크 수신 → [참여 여부 투표 (/room/[id]/vote)]
        ├── "안 먹어요" → 홈으로 (해당 약속 불참 표시)
        └── "먹을게요!" → [식당 추천 목록 (/room/[id]/restaurants)]
              └── 식당 카드 클릭
                    │
                    ├── [투표 모드] → [식당 상세 - 투표 (/restaurant/[id]?mode=vote)]
                    │     ├── "이 식당 추천해요!" → 투표 수 반영 + 실시간 현황 표시
                    │     └── 투표 마감 (리더가 마감 or 전원 투표 완료)
                    │           └── 최다 득표 식당 자동 확정 → [메뉴 선택]
                    │
                    └── [리더 선택 모드] → [식당 상세 - 리더 (/restaurant/[id]?mode=leader)]
                          ├── 일반 팀원: "여기서 먹어요?" 버튼 비활성화 (리더 대기 중 표시)
                          └── 리더: "여기서 먹어요!" 버튼 활성화 → 즉시 확정
                                └── [메뉴 선택 (/room/[id]/menu)]
                                      └── 메뉴 선택 완료 → [최종 요약 (/room/[id]/summary)]
```

### 7A. 식당 상세 — 투표 모드 (`/restaurant/[id]?mode=vote`)

모든 팀원이 이 화면에서 원하는 식당에 투표할 수 있어요.

```
┌──────────────────────────────┐
│  ← 식당 정보       🗳️ 투표중  │
├──────────────────────────────┤
│  [ 식당 사진 ]               │
│                              │
│  상암 돼지갈비               │
│  ⭐ 4.5 (리뷰 128개)         │
│  📍 서울 마포구 상암동 123    │
│  🕐 11:00 - 21:00            │
│  ☎ 02-1234-5678              │
│                              │
│  ─── 주요 메뉴 ───           │
│  • 삼겹살 1인분 ........ 13,000 │
│  • 목살 1인분 ......... 14,000 │
│  • 된장찌개 ........... 8,000 │
│                              │
│  ─── 현재 투표 현황 ───      │
│  👍 홍길동, 김철수 (2표)      │
│  아직 투표 안 한 팀원: 이영희  │
│                              │
│  ┌────────────────────────┐  │
│  │  👍 이 식당 추천해요!   │  │  ← 모든 팀원 누를 수 있음
│  └────────────────────────┘  │
│  (이미 투표했다면 "투표 취소"로 변경) │
└──────────────────────────────┘
```

> **투표 마감 처리:**
> - 리더가 식당 추천 목록에서 "투표 마감" 버튼을 누르거나
> - 전원이 투표를 완료하면 자동 마감
> - 동점일 경우 리더가 최종 결정

### 7B. 식당 상세 — 리더 선택 모드 (`/restaurant/[id]?mode=leader`)

리더만 식당을 확정할 수 있고, 팀원들은 리더의 선택을 기다려요.

```
┌──────────────────────────────┐
│  ← 식당 정보      👑 리더 선택 │
├──────────────────────────────┤
│  [ 식당 사진 ]               │
│                              │
│  상암 돼지갈비               │
│  ⭐ 4.5 (리뷰 128개)         │
│  📍 서울 마포구 상암동 123    │
│  🕐 11:00 - 21:00            │
│  ☎ 02-1234-5678              │
│                              │
│  ─── 주요 메뉴 ───           │
│  • 삼겹살 1인분 ........ 13,000 │
│  • 목살 1인분 ......... 14,000 │
│  • 된장찌개 ........... 8,000 │
│                              │
│  [리더일 때]                 │
│  ┌────────────────────────┐  │
│  │  👑 여기서 먹어요! 확정  │  │  ← 리더만 활성화
│  └────────────────────────┘  │
│                              │
│  [일반 팀원일 때]             │
│  ┌────────────────────────┐  │
│  │  리더가 선택 중이에요...│  │  ← 비활성화 (회색)
│  └────────────────────────┘  │
└──────────────────────────────┘
```

> **팀원 경험:** 식당 상세는 볼 수 있지만 확정 버튼은 리더만 누를 수 있어요. 리더가 확정하면 모든 팀원에게 푸시 알림 발송 → 메뉴 선택 화면으로 자동 이동.

## MVP 권장 순서

처음부터 모든 화면을 만들면 힘드니, 아래 순서로 단계적으로 만드는 걸 추천해요:

**1단계 (핵심 MVP):** 온보딩 → 밥약속 만들기 → 참여 투표 → 최종 요약
**2단계 (식당 기능):** 식당 추천 목록 → 식당 상세 → 메뉴 선택
**3단계 (완성):** 홈 화면 (이력 관리) → 초대 공유 → 카카오 공유
